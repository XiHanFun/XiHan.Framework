# XiHan.Framework.Application

## 项目定位

`XiHan.Framework.Application` 承载应用层实现基础能力，用于组织用例、事务边界、输入输出协调和跨领域流程编排。

## 责任边界

本项目负责：

1. 应用服务基类或公共实现约定
2. 用例编排
3. 面向外部契约的实现层支撑

本项目不负责：

1. HTTP、gRPC、SignalR 等协议适配
2. ORM、缓存、消息等基础设施实现
3. 核心领域规则本身

## 依赖方向

本项目依赖应用契约与领域抽象，但不依赖具体宿主或具体 Provider。

## 当前阶段

当前项目已建立应用服务基础类型，后续将继续补充事务、验证、授权、本地化等横切能力接入点。
