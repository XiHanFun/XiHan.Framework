# XiHan.Framework.Bootstrap.Abstractions

## 项目定位

`XiHan.Framework.Bootstrap.Abstractions` 用于承载框架引导层抽象。

它位于最小内核之上、具体运行时实现之下，专门描述应用创建、应用生命周期和可选模块运行时所需的公共契约。

## 责任边界

本项目负责：

1. 应用信息访问器与应用抽象
2. 应用初始化与关闭上下文
3. 可选模块生命周期接口
4. 模块描述、模块加载与模块管理抽象
5. 可插拔的程序集发现与类型发现抽象

本项目不负责：

1. 具体应用工厂实现
2. 模块拓扑排序与运行时编排
3. 具体程序集扫描策略
4. 业务规则和基础设施实现

## 依赖方向

本项目依赖 `XiHan.Framework.Kernel`，用于在不污染最小核心的前提下承接运行时与宿主相关的稳定抽象。

所有需要参与应用引导链路的项目，应优先依赖本项目而不是反向扩充 `Kernel`。

## 当前阶段

当前项目用于承接从旧 `Core` 中拆出的应用生命周期接口、运行时上下文和发现机制抽象。

后续若新增引导层扩展点，优先放在本项目或其实现层，不再回流到 `Kernel`。
