# XiHan.Framework.Metadata

## 项目定位

`XiHan.Framework.Metadata` 用于承载框架自身的元数据、版本信息、组织信息和运行时入口程序集摘要。

## 责任边界

本项目负责：

1. 框架名称与显示名称
2. 作者、组织、仓库、文档、许可证等元信息
3. 框架版本摘要与运行时入口程序集信息

本项目不负责：

1. 模块化机制
2. 领域模型
3. 应用服务
4. 基础设施实现

## 依赖方向

本项目尽量只依赖 BCL，不依赖其他框架项目，作为最轻量的元数据提供层。

## 当前阶段

当前项目已迁移旧 `XiHan.Framework.Metadata` 的核心静态元数据能力，后续可继续补充构建信息和发布通道元数据。
