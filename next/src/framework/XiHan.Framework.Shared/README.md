# XiHan.Framework.Shared

## 项目定位

`XiHan.Framework.Shared` 用于承载跨层共享、但不属于框架内核机制的数据模型和基础契约。

## 责任边界

本项目负责：

1. 通用结果模型
2. 分页模型
3. 排序与筛选基础模型
4. 跨层共享常量和错误码

本项目不负责：

1. 模块化和生命周期机制
2. 领域实体与领域规则
3. 基础设施抽象与实现
4. 与具体业务模块绑定的 DTO

## 依赖方向

本项目只允许依赖 `XiHan.Framework.Kernel` 或更底层的 BCL，不依赖应用层和基础设施层。

## 当前阶段

当前项目已提供最小共享模型骨架，后续将逐步补充分页结果、排序模型、错误码与通用响应模型。
