# XiHan.Framework.Domain

## 项目定位

`XiHan.Framework.Domain` 承载领域抽象层，用于描述仓储、规约、领域服务等领域契约。

## 责任边界

本项目负责：

1. 仓储接口
2. 规约接口
3. 领域服务接口
4. 基于领域原语之上的通用领域契约

本项目不负责：

1. 实体和值对象原语
2. 仓储实现
3. ORM、缓存、消息中间件等技术实现
4. 宿主和传输协议适配

## 依赖方向

本项目依赖 `Domain.Primitives`、`Kernel` 与 `Shared`，但不得依赖应用实现、宿主项目和具体 Provider。

## 当前阶段

当前项目已开始承载仓储契约，后续将继续补充规约、领域服务与领域事件协作相关抽象。
