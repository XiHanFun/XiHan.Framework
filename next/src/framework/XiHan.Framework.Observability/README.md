# XiHan.Framework.Observability

## 项目定位

`XiHan.Framework.Observability` 用于承载日志、指标、追踪等可观测性能力的统一接入面。

## 责任边界

本项目负责：

1. 可观测性统一入口
2. 指标、追踪与日志之间的协同约定
3. 后续 OpenTelemetry 等能力的接入准备

本项目不负责：

1. 具体平台适配实现
2. 业务指标定义本身

## 依赖方向

本项目依赖日志基础能力，但尽量不依赖重型第三方实现。

## 当前阶段

当前项目已建立可观测性选项骨架，后续将继续补充指标、追踪和 OpenTelemetry 接入面。
