# XiHan.Framework.MultiTenancy.Abstractions

## 项目定位

`XiHan.Framework.MultiTenancy.Abstractions` 定义多租户能力的核心抽象，用于隔离租户上下文、租户解析和租户访问约束。

## 责任边界

本项目负责：

1. 当前租户上下文抽象
2. 租户解析器抽象
3. 租户标识约定

本项目不负责：

1. HTTP 请求中的租户解析实现
2. 数据库级租户隔离实现
3. 具体租户管理业务

## 依赖方向

本项目只依赖内核，不依赖任何宿主和基础设施实现。

## 当前阶段

当前项目已建立当前租户上下文抽象，后续将继续补充租户解析器和租户切换相关契约。
