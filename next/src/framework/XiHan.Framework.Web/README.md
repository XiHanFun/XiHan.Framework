# XiHan.Framework.Web

## 项目定位

`XiHan.Framework.Web` 是框架面向 Web 场景的基础构建块，用于沉淀与具体宿主无关的 Web 通用能力。

## 责任边界

本项目负责：

1. Web 层基础扩展点
2. 与应用层衔接的通用适配约定
3. 与本地化、验证、授权等 Web 横切能力的统一接入点

本项目不负责：

1. 具体 API 宿主启动逻辑
2. 具体协议端点实现
3. 业务模块接口本身

## 依赖方向

本项目依赖应用层与必要抽象，但不依赖具体宿主项目。

## 当前阶段

当前项目已建立 Web 层最小扩展点，后续将继续补充过滤器、中间件和统一错误处理接入面。
