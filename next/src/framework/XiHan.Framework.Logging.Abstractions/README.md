# XiHan.Framework.Logging.Abstractions

## 项目定位

`XiHan.Framework.Logging.Abstractions` 用于定义框架级日志契约，隔离日志上下文、日志分类和框架统一日志语义。

## 责任边界

本项目负责：

1. 框架日志写入抽象
2. 框架日志分类约定
3. 统一日志上下文契约

本项目不负责：

1. 控制台日志实现
2. Serilog 实现
3. 具体日志平台接入细节

## 依赖方向

本项目只依赖 `Kernel` 和 BCL，是所有日志实现项目的共同上游。

## 当前阶段

当前项目已建立最小日志抽象，后续将继续补充日志上下文、分类、审计日志等框架级语义。
