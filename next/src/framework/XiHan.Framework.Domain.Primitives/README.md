# XiHan.Framework.Domain.Primitives

## 项目定位

`XiHan.Framework.Domain.Primitives` 用于承载领域层中最基础、最稳定、最不应混入基础设施语义的领域原语。

## 责任边界

本项目负责：

1. 实体抽象
2. 值对象抽象
3. 聚合根标记抽象
4. 领域事件基础模型

本项目不负责：

1. 仓储接口
2. 规约接口
3. 领域服务接口
4. 任何数据访问或基础设施实现

## 依赖方向

本项目只依赖 `Kernel` 和 BCL，作为领域相关项目的更底层基础。

## 当前阶段

当前项目已开始承载实体原语，后续将继续补充值对象、聚合根和领域事件等基础领域概念。
