# XiHan.Framework.Localization

## 项目定位

`XiHan.Framework.Localization` 提供国际化默认实现。

## 责任边界

本项目负责：

1. 对接微软本地化基础设施
2. 提供默认字符串本地化器适配
3. 为上层应用和宿主提供统一接入点

本项目不负责：

1. 具体业务模块的资源内容定义
2. 宿主层请求文化解析策略

## 依赖方向

本项目依赖 `Localization.Abstractions`，并优先采用微软官方本地化包。

## 当前阶段

当前项目提供默认占位实现，后续将逐步接入资源文件、虚拟文件系统和其他资源来源。
