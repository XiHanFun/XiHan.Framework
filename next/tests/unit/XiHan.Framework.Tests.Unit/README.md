# XiHan.Framework.Tests.Unit

## 项目定位

`XiHan.Framework.Tests.Unit` 用于承载框架和模块的单元测试。

## 责任边界

本项目负责：

1. 最小行为验证
2. 纯内存测试
3. 快速回归验证
4. 架构边界守护测试

本项目不负责：

1. 集成测试
2. 性能测试
3. 正式打包和发布

## 生命周期约束

本项目属于测试资产，只参与测试构建和测试执行，不参与正式发布流程。

## 当前阶段

当前项目已建立最小单元测试骨架，并开始补充：

1. 核心行为回归测试
2. 架构边界守护测试
3. 核心项目归属守护测试

后续会继续增加更多分层约束，确保 `Kernel / Bootstrap / Host` 的职责不会再次漂移。
